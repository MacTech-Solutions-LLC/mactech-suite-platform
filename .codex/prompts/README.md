# Codex Prompt Library

This folder contains reusable prompts for safe, repeatable repository work.

## Prompts

- `review-current-changes.md` - review local changes before commit or pull
  request
- `create-small-slice.md` - plan and implement a small, traceable change
- `security-safe-editing.md` - apply security-safe editing constraints

Prompts should avoid secrets, customer data, credentials, tokens, and unsupported
production or certification claims.
